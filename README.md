# 📒 Contact Book (Python CLI Project)

A simple and efficient **Contact Book application** built using Python.  
This project allows users to store, manage, and retrieve contact details such as name, phone number, and email using a JSON file for persistent storage.

---

## 🚀 Features

- ➕ Add new contacts
- 📋 View all contacts
- 🔍 Search contact by name
- ✏️ Update existing contact details
- ❌ Delete contact
- 💾 Save contacts to a JSON file
- 📂 Load contacts automatically on startup

---

## 🛠️ Technologies Used

- Python
- JSON (for data storage)
- Object-Oriented Programming (OOP)
